Situation

Alex has refused to complete his homework for over a week. His parents have tried arguing, forcing, and removing privileges. Nothing has worked.

Your Objective

  • Discover why Alex refuses homework.
  • Convince him to willingly begin.

Mission Rules

  • 25 Messages
  • No dialogue options
  • Every word matters
  • No correct answers

AI Settings

Gemini API Key

About

CHANGING MINDS uses Google's Gemini AI to power every conversation. To keep this prototype free for everyone, you'll use your own Gemini API key instead of ours.

Privacy

Your API key is stored only in your browser and is used only to communicate directly with Google's Gemini API. CHANGING MINDS does not collect, store, or transmit your API key.

Recommendation

We recommend creating a separate free Gemini API key specifically for CHANGING MINDS testing. If you're using a shared computer, you can remove it after you're finished.
